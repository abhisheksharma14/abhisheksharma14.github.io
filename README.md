## üì¶ Installation

1. Clone the repository
```bash
git clone <your-repo-url>
cd portfolio-website
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

The site will be available at `http://localhost:5173/`

## üöÄ Deployment

### Build for Production

```bash
npm run build
```

This will create an optimized production build in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

### Deployment Options

You can deploy this portfolio to various platforms:

#### Vercel
```bash
npm install -g vercel
vercel
```

#### Netlify
```bash
npm install -g netlify-cli
netlify deploy
```

#### GitHub Pages
1. Update `vite.config.js` to include your repository name as base
2. Build and deploy:
```bash
npm run build
# Deploy the dist folder to gh-pages branch
```

## üìù Customization

To customize this portfolio for your own use:

1. Update personal information in the components:
   - `Hero.jsx` - Name and introduction
   - `About.jsx` - About section and social links
   - `Experience.jsx` - Work experience
   - `Projects.jsx` - Project details
   - `Skills.jsx` - Skills, certifications, and education
   - `Contact.jsx` - Contact information

2. Update color scheme in CSS files if desired
3. Replace profile placeholder with your photo in `About.jsx`
